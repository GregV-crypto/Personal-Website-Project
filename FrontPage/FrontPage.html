The Investor

Taking the best out of two Interests

The purpose of the website is to measure my progess as I grow as a website developer. It's functional purpose however is to be that of a basic investment website built from the ground up built in pure HTML, CSS, and Javascript.

JavaScript

JavaScript is used all of the background operations. Everything from creating the user's personal list of investments, to fetching data from APIS for each webpage. Simulating a real world investment website is the goal.

HTML-CSS

HTML and CSS are of course used for the website's actual creation and styling.
